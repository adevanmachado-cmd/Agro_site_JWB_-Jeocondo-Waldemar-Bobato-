# 🌱 Consciência no Campo — Segurança no Trabalho Rural

Uma página web responsiva focada na conscientização sobre a prevenção de acidentes de trabalho na agricultura paranaense. O projeto apresenta dados estatísticos do setor, diretrizes de segurança baseadas na NR-31 e integra um formulário de pesquisa para mapear a percepção de risco no campo.

---

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização moderna com variáveis globais, Grid Layout, Flexbox e foco em responsividade.
* **Google Forms:** Integração de formulário externo para coleta de dados de pesquisa.

---

## 📋 Funcionalidades Principais

* **Seção Estatística:** Cards dinâmicos destacando os principais riscos do agronegócio no Paraná.
* **Guia de Diretrizes:** Lista de boas práticas de prevenção alinhadas com as normas regulamentadoras.
* **Responsividade:** Layout totalmente adaptável para dispositivos móveis e desktops.
* **Chamada para Ação (CTA):** Botão estilizado com efeitos de transição que direciona o usuário para a pesquisa de campo.

---

## 📂 Estrutura do Código

O projeto foi construído em um arquivo único para facilitar a portabilidade:

* **`:root` CSS:** Centralização da paleta de cores (verde agrícola e vermelho de alerta).
* **`main` semântico:** Divisão clara entre introdução, dados, prevenção e ação.
* **Media Queries:** Ajustes de tipografia e espaçamento para telas menores que 600px.

---

## 🔧 Como Executar o Projeto

1. Baixe ou clone este repositório.
2. Certifique-se de que a estrutura do arquivo está salva como `index.html`.
3. Dê um duplo clique no arquivo `index.html` para abri-lo em qualquer navegador web.

---

## 📝 Nota de Configuração

> 💡 **Importante:** No código atual, a tag do botão de pesquisa está apontando para o link de edição do formulário. Lembre-se de substituir o link no atributo `href` caso queira utilizar a versão pública de resposta do seu Google Form:
> ```html
> <a href="SEU_LINK_AQUI" target="_blank" class="btn-google-form">Acessar Pesquisa no Google Forms</a>
> ```
